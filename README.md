An automated Python script that organizes your folders by file extension.

## Features
- Automatically moves PDFs to "Documents", JPGs to "Images", etc.
- Runs on a schedule every day at a set time.

## Setup
1. Install requirements: `pip install -r requirements.txt`
2. Edit `target_dir` in `src/main.py` to your folder path.
3. Run: `python src/main.py`
